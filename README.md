## Описание проекта
Мы - команда аналитиков, выполняющая проект по обработке и анализу данных из разных источников (LitRes, Litnet, Author Today). Наша задача — продемонстрировать навыки веб-скрапинга, работы с API, обработки полученных данных, а также EDA анализа.

### Заказчик / пользователь результатов
Основной заказчик: платформы самиздата (например, LitRes) и онлайн-магазины, работающие с книгами независимых авторов.
Дополнительные пользователи: сами авторы, которые хотят понять, какие характеристики книги влияют на её популярность и успешность публикации.

### Бизнес-цель
Проект направлен на анализ книг независимых авторов на платформах самиздата с целью выявления закономерностей между характеристиками книг (жанр, автор, текст аннотации и др.) и их популярностью. Результаты анализа помогут платформам оптимизировать рекомендации и продвижение новых книг, а авторам — принимать обоснованные решения о публикации и улучшении своих произведений.

## Описание данных
### LitRes
• url - URL страницы книги

• title - название книги  

• authors - авторы

• rating - рейтинг книги

• rating_count - количество оценок

• reviews_count - количество отзывов

• price - цена

• genres - жанры

• age_limit - возрастное ограничение

• release_date - дата выхода на литрес

• written_date - дата написания

• pages - объем в страницах

• isbn - ISBN

• copyright_holder - правообладатель

• formats - доступные форматы

• description - описание книги

Обработано 500 страниц, 11973 книг7

### Author Today
В датасет включены поля:

• title - название книги

• authors - авторы

• categories - жанры/категории

• date - дата публикации

• symbols_count - количество символов

• a4_sheets - объем в печатных листах

• views - количество просмотров

• likes - количество лайков

• comments - количество комментариев

• reviews - количество рецензий

• price - цена

• cycle - принадлежность к циклу

• exclusive - эксклюзивность

• annotation - аннотация книги

Обработано 9,975 уникальных книг, 14 параметров, 400+ страниц.

### Litnet API
В датасет включены поля:

•	id — уникальный идентификатор книги

•	available - статус доступности

•	rate — рейтинг книги

•	title — название

•	author — автор

•	genre — жанр (преобразованный из id в текст)

•	price — цена

•	year — год публикации

•	description — описание книги

•	age — возрастное ограничение

Датасет на 5 000 строк сохранён в форматах JSON и CSV (data/raw/litnet_books.json и data/raw/litnet_books_dataset.csv).

### Итоговый датасет
• title — название книги

• authors — автор книги

• genre — жанры книги (через запятую или список)

• description — аннотация книги (описание сюжета)

• desc_len — длина аннотации в символах

• author_books_count — количество опубликованных книг у автора

• is_series — принадлежит ли книга к серии (1 — да, 0 — нет)

• views_authoday — количество просмотров книги на Author.Today

• likes_authoday — количество лайков на Author.Today

• comments_authoday — количество комментариев

• reviews_authoday — количество отзывов пользователей

• price_authoday — цена книги на Author.Today, ₽

• year_authoday — год публикации книги на Author.Today

• rating_litres — средний рейтинг книги на ЛитРес

• rating_count_litres — количество оценок на ЛитРес

• reviews_count_litres — количество отзывов на ЛитРес

• price_litres — цена книги на ЛитРес, ₽

• year_litres — год выхода книги на ЛитРес

• price_litnet — цена книги на ЛитНет, ₽

• year_litnet — год выхода книги на ЛитНет

• has_litres — флаг наличия книги на ЛитРес (1 — есть, 0 — нет)

• has_litnet — флаг наличия книги на ЛитНет

• has_authoday — флаг наличия книги на Author.Today

• cycle_authoday — название серии или цикла, если книга является её частью

# EDA Анализ
Проведена очистка данных из объединённого датасета (Litres, AuthorToday, LitNet).

 • Выполнен исследовательский анализ: проверка пропусков, распределений и корреляций.
 
 • Проведена нормализация жанров и анализ их частоты.
 
 • Построены визуализации для ключевых метрик — рейтингов, цен и активности авторов.
 
 • Определены закономерности между популярностью книг, оценками и ценой.
 
 • Сформулированы направления для бизнес-применения датасета (например, рекомендация цен и подбор авторов по метрикам вовлечённости).
 
# Выводы

1. Опытные авторы получают больше вовлечённости.
Количество лайков и цена книги растут с числом публикаций — активные авторы формируют аудиторию и могут повышать стоимость.

2. Популярность влияет на монетизацию.
Чем больше книг у автора, тем ниже доля бесплатных — опытные авторы чаще монетизируют контент.

3. Качество контента стабилизировалось.
Средний рейтинг книг после 2000-х стабильно держится на высоком уровне (около 4.5–5).

4. Рынок Author.Today дорожает.
Медианные цены на книги растут ежегодно, особенно после 2020 года. Это отражает укрепление платформы и рост готовности аудитории платить.

5. Короткие аннотации эффективнее для книг высокого ценового сегмента.
По тепловой карте видно, что краткие описания у более дорогих книг коррелируют с наибольшим числом лайков — пользователи ценят чёткие и лаконичные тексты.

6. Кластеры авторов различаются по стратегии.
• Кластер 0 — начинающие, много бесплатных книг, низкие цены.
• Кластер 1 — коммерческие авторы со средними метриками.
• Кластер 2 — топ-авторы с высокой ценой, популярностью и объёмом публикаций.
