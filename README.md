## Описание проекта
Мы — команда аналитиков, выполняющая проект по обработке и анализу данных из разных источников (LitRes, Litnet, Author Today). Наша задача — продемонстрировать навыки веб-скрапинга, работы с API, обработки полученных данных, а также EDA анализа.

### Заказчик / пользователь результатов
Основной заказчик: платформы самиздата (например, LitRes) и онлайн-магазины, работающие с книгами независимых авторов.
Дополнительные пользователи: сами авторы, которые хотят понять, какие характеристики книги влияют на её популярность и успешность публикации.

### Бизнес-цель
Проект направлен на анализ книг независимых авторов на платформах самиздата с целью выявления закономерностей между характеристиками книг (жанр, автор, текст аннотации и др.) и их популярностью. Результаты анализа помогут платформам оптимизировать рекомендации и продвижение новых книг, а авторам — принимать обоснованные решения о публикации и улучшении своих произведений.
